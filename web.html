WEB DEVELOPMENT TASK

Build a Multi-Section Blog Page

Objective: Create a multi-section blog page using HTML to demonstrate your web development skills, including formatting content, creating links, embedding media, and organizing information.

TASK REQUIREMENTS
  • Follow HTML5 structure: <!DOCTYPE html>, <html>, <head>, <body>
  • Usage of heading tags (h1 to h6), paragraphs, bold, italic, marquees, subscript, superscript, button, image tag <img> with src and alt, table, lists (unordered, ordered, description)
  • Several links: internal (within page), external, downloadable, linking to page (id & #)
  • Embedded media tags: audio, video, iframee
  • Sectioned content: welcome, table of topics, links to read further, media section with a video and audio clip
INSTRUCTIONS
1

Set up the HTML5 structure: Start by declaring <!DOCTYPE html>, <html>, <head>, <body>.

2

Format the Content: Define a header, add headings, paragraphs, and use formatting like bold, italic, buttons, and marquee.

3

Create Links & Lists: Add unordered, ordered lists and include internal and external links.

4

Add Images, Tables & Media: Insert images, create tables, and embed video, audio and iframe.

SUBMISSION CHECKLIST
  • Use proper HTML5 structure with DOCTYPE, html, head, and body to ensure the webpage works correctly in all browsers.
  • Include headings, paragraphs, formatting elements, buttons, and marquee to organize and style the content clearly.
  • Add lists, table, images, video, audio, and iframe to make the webpage interactive and visually appealing.
